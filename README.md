# 基于循环神经网络的合肥地区安徽大学磬苑校区气温预测
本项目是深度学习课程的时间序列预测项目，实现了基于RNN/LSTM/GRU的天气时序预测，通过多超参数对比试验与数据量消融实验验证了超参数与数据量对模型性能的影响，并对比了不同模型在训练效率与预测精度上的平衡。

## 功能特点
- 只支持单步天气预测（由过去两周预测未来1天）
- 支持多变量联合预测（温度、降水、风速等气象要素）
- 内置RNN、LSTM、GRU三种基础模型，可通过配置文件快速切换
- 实现了早停、权重衰减、Dropout等正则化方法防止过拟合
- 提供完整的训练日志与可视化结果

## 模型说明
本项目实现了三种经典的循环神经网络模型，并进行了针对性优化：
1.  **基础RNN**：最简单的循环神经网络，作为基准模型
2.  **LSTM**：长短期记忆网络，解决了RNN的梯度消失问题
3.  **GRU**：门控循环单元，在保持LSTM性能的同时大幅提升训练效率

## 实验设置
- **随机种子**：4044，基于该随机种子可复现训练过程与训练结果
- **数据集**：基于Open-Meteo ERA5再分析数据，构建了3年、6年、30年三个时间跨度的数据集
- **输入特征**：过去14天的11个气象变量+2个时间特征
- **预测目标**：未来1天的日平均温度（可扩展为多步多变量预测）
- **训练参数**：最大迭代次数100，早停耐心值15，权重衰减1e-5，Dropout率0.2

## 核心结论
1.  **数据量的影响远大于模型架构微调**：30年数据集的预测精度比3年数据集提升了47%
2.  **GRU是综合表现最优的模型**：在训练速度比LSTM快30%的同时，预测精度基本持平
3.  **多变量联合预测优于单变量预测**：加入降水、风速等变量后，温度预测误差降低了12%

## 使用方法
### 1. 环境安装

pip install -r requirements.txt


### 2. 数据准备
本项目数据来自Open-Meteo Archive API，可通过以下方式获取：
from src.data import download_data
download_data(start_date="1996-05-13", end_date="2026-05-13")


### 3. 模型训练

python src/train.py --config configs/gru_base.yaml
